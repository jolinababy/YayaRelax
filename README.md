# 小鸭梨 · YayaRelax
一款用音乐舒缓情绪、减轻压力的轻量心理治愈应用。

## 🌟 介绍
小鸭梨以温柔音乐为核心，帮助用户在忙碌生活中放松身心，缓解焦虑与压力，陪伴你找回内心的平静与力量。

## ✨ 主要功能
- 🌿 **正念冥想**：专业引导式冥想，配合舒缓音乐，快速放松身心
- 📝 **减压日记**：记录情绪与日常，释放压力，梳理思绪
- 📖 **答案之书**：迷茫时刻随机获取治愈短句，给予温暖指引
- 🎵 **舒缓音乐**：专注助眠、减压、放松的纯音乐环境

## 🎯 适用场景
- 工作学习压力大、情绪焦虑
- 睡前难以入睡、需要放松
- 心情低落、需要治愈与鼓励
- 日常冥想、情绪管理
